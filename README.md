# I3VTimer

MRE for Altspace VR to display a timer in the world

## Usage Instructions

Install Node.js 8.12 or newer, which includes NPM 6.4.1 or newer, from nodejs.org

From command prompt:

git clone http://github.com/contactnathan/i3vtimer
cd i3vtimer
npm install
npm run build
npm start
